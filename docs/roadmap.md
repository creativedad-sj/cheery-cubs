# Roadmap

## Product Direction
Cheery Cubs is moving from a hobby app toward a real launch product.

The roadmap should stay focused on:

- clear parent trust
- meaningful age and skill guidance
- distinct game mechanics
- quality over raw game count

## Phase 1: Launch Foundation
Goal: make the app feel structured, real, and ready for serious testing.

Focus:

- parent-first discovery
- stage-based browsing
- starter baby co-play section
- strong core catalog
- stable deploy/build flow
- first documentation set

Key outputs:

- home and browse flow
- stage metadata
- skill metadata
- core launch games
- docs and handoff notes

## Phase 2: Category Strength
Goal: make each top-level category feel credible and balanced.

Focus:

- fill the weakest categories first
- especially the next distinct gaps after the new memory wave
- reduce duplicate-feeling games
- improve quality on the most played games

Recommended build priority:

1. `World Around Me`
2. `Baby Co-Play`
3. `Letters & Reading`
4. `Numbers & Counting`
5. `Thinking & Patterns`
6. polish and duplicate reduction across the catalog

Suggested Phase 2 additions:

- `Visual Search`
- `Animal Sounds`
- `Touch the Light`
- `Bubble Reach`
- `Body Parts`
- `Food Match`
- `Home Helpers`
- `Before and After`
- `Upper and Lower Match` polish pass
- `Follow Path` polish/expansion

Current note:

- `Visual Search` is now built
- Phase 2 `World Around Me` has started with `Body Parts`, `Food Match`, and `Home Helpers`
- the next work should still favor distinct mechanics and better coverage over raw volume

## Phase 3: 60-Game Core Catalog
Goal: reach a balanced 60-game catalog across 6 top-level categories.

Target:

- `10` games per category
- `60` total games

Categories:

- `Baby Co-Play`
- `Letters & Reading`
- `Numbers & Counting`
- `Thinking & Patterns`
- `Memory & Focus`
- `World Around Me`

Important rule:

- do not reach 60 by padding the catalog with near-duplicates
- each game should feel mechanically or developmentally distinct

## Beyond Phase 3
Once the 60-game core is strong, possible next product tracks:

- dashboard insights by skill
- guided learning paths
- premium subscription model
- more polished visual asset system
- app store onboarding and screenshots
- curriculum-based recommendation engine
