# Core Catalog Plan

## Purpose
This file answers the practical roadmap question:

- what counts toward the real `60-game` product
- what already exists
- what is still missing by category
- which games are `Not Started`

Status meanings:

- `Core Live`: implemented and counted toward the `60-game` target
- `Extra Live`: implemented, playable, but not counted toward the core target
- `Not Started`: planned but not yet built

## Core Catalog Goal

- `6` top-level categories
- `10` distinct core games in each category
- `60` total core games

Current snapshot:

- `49` implemented games total
- `40` current core games
- `9` current extra games
- `20` core games still to build
