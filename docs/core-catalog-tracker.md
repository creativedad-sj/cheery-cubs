# Core Catalog Tracker

## Goal
Build a `60-game` core catalog with:

- `6` top-level categories
- `10` distinct games in each category

Detailed category-by-category status lives in `docs/core-catalog-plan.md`.

Distinct means:

- different mechanics, developmental goals, or play feel
- not just the same matching shell with different nouns
- each category should include both simple entry games and richer interactive games
